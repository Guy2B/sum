# V1.6.1 test checklist

## Context

- Save a primary goal and constraints.
- Select active days and cross-analysis permissions.
- Add a weekly check-in.
- Verify that the context score increases.
- Ask Σ to guide you using the whole context.

## Unified Mail

- Enter a Gmail, Outlook, Yahoo and GMX address in demo mode.
- Verify automatic provider detection.
- Test a professional domain and select a host.
- Create a task and a reply reminder from a message.
- Disconnect an address.

## Local intelligence

- Open the AI settings from the Coach header.
- Verify the guided engine remains active.
- Enable semantic understanding on a networked test device and wait for the first model download.
- Ask the same question with several phrasings and check intent routing.
- Enable Chrome local rewrite only on a compatible computer.
- Disable both optional layers and confirm the Coach still responds.

## Regression

- Four languages.
- Light and dark modes.
- Five editions.
- Mobile navigation.
- PWA refresh and offline shell.
- Free and owner-preview modes.
