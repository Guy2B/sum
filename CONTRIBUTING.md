# Contributing

This is a proprietary project. Contributions require prior approval from Al.G.B.r.

Before opening a pull request:

1. Create a feature branch.
2. Run `npm run check`.
3. Test Free, Pro preview, all four languages, light/dark mode, and the relevant edition.
4. Do not commit secrets or real user data.
5. Document user-visible changes in `CHANGELOG.md`.
