SUM support

Most questions can be resolved without waiting for support.

How do I start?

Open the application, complete the one-minute setup, add one task and one learning goal, then run the Smart Coach analysis.

Where are my data stored?

By default, in the current browser on the current device. Use Backup & Sync → Export JSON regularly. Pro users can optionally enable encrypted secure backup.

Why can I not change language?

SUM Free keeps the language selected during setup. SUM Pro unlocks all four interface languages.

How does the Smart Coach work without an API?

It calculates signals from tasks, projects, cash flow, wellbeing and learning, applies prioritised rules, and generates translated recommendations locally.

How do I activate Pro?

After purchase, enter the purchase email and licence key under Account & Plan. A local demo key works only on local development installations.

How do I move to another device?

Export a JSON backup, install or open SUM on the new device, then import the file. Cloud restore is available when the Pro backend is configured.

Charts are missing offline

Open SUM once while connected so the chart library can be cached by the PWA service worker. All core data entry functions work without charts.

Contact

Include the SUM version, browser, device type, language, steps that caused the problem and a screenshot without private journal or finance content.
