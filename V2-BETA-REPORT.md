# Rapport V2.0 Beta Google Cloud Edition

Base migrée vers Firebase/Google Cloud. La version contient une infrastructure déployable, mais elle n'est opérationnelle en ligne qu'après création d'un projet Firebase et remplacement des valeurs `REPLACE_WITH_...`.

Validé dans le paquet: configuration Firebase, client Auth/Firestore/Storage/Functions, règles propriétaires, index, stockage privé, fonctions santé/Premium/feedback, correctifs globaux d'espacement des titres et documentation de déploiement.

Non inclus sans comptes externes: secrets OAuth, autorisations Gmail/Calendar/YouTube/Meta, Apple Health natif signé, paiement Stripe live.
