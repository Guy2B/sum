# Sigma Life OS V4.7 — Google Complete

## Réellement intégré

- Une connexion OAuth Google groupée avec autorisations progressives.
- Gmail : import et envoi.
- Calendar : lecture, création, mise à jour et suppression via `SigmaGoogle`.
- Drive : sauvegarde privée dans `appDataFolder`.
- YouTube : import des abonnements.
- Google Contacts : import via People API.
- Google Maps : carte intégrée, recherche de lieux et itinéraires.
- Un seul panneau de compte Google avec test API réel service par service.

## APIs à activer

Gmail API, Google Calendar API, Google Drive API, YouTube Data API v3, People API, Maps JavaScript API, Places API et Directions API.

Un service n'est affiché « Connecté » qu'après un appel API réussi.
