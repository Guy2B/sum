# Direct social connectors V1.7

## Adapter status

- Meta: Instagram professional and Facebook Page OAuth adapter already present.
- YouTube: OAuth and channel comment adapter already present.
- X: OAuth 2.0 PKCE and mention adapter source added.
- LinkedIn: OpenID profile connection and approved-product boundary added.
- TikTok: OAuth, user profile and recent-video signal adapter added.

## Important boundary

An adapter in the repository does not mean that a provider has granted production access. Provider plan, scopes, account type, review and regional policy determine real capabilities. The frontend therefore displays capability states and never promises universal private-message access.

## Safety defaults

V1.7 reads and ranks authorised interactions. It creates local tasks, reminders and drafts. It does not automatically send messages, publish, delete, hide, block or moderate external content.
