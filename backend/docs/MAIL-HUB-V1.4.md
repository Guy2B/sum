# SUM Mail Hub — V1.5

V1.5 introduces a provider-neutral, local-first Mail Hub. Without a connector backend, Gmail, Outlook, Yahoo and GMX run in an explicit demo mode. No real credentials are requested or stored by the browser.

Features:
- multiple mailbox cards;
- unread, important and suggested-reply counts;
- message-to-task conversion;
- filters by account and attention state;
- local heuristic classification;
- dashboard summary.

Real provider connections are added in V1.5 through the optional secure backend.
