# Sigma Life OS V4.8.1 — Multi Providers amélioré

Inclut toute la V4.7.1, plus :

- Microsoft / Outlook via OAuth Microsoft Graph.
- Hub par comptes pour Microsoft, Yahoo, GMX, iCloud et IMAP générique.
- Diagnostic administrateur unifié Google + autres fournisseurs.
- Statuts honnêtes pour les connecteurs serveur non déployés.
- Cache-busting `v=4810`.

## Mise à jour minimale GitHub depuis V4.7.1

Téléverser/remplacer :

- `app.html`
- `style.css`
- `platform-config.js`
- `modules/microsoft-graph.js`
- `modules/multi-provider-hub-v48.js`
- `modules/system-diagnostics.js`
- `service-worker.js`

Conserver vos vraies valeurs dans `firebase-config.js`, `google-cloud-config.js` et `platform-config.js`.
