# Sigma V4.9.7 — X Setup

1. Créer un projet et une application dans le portail développeur X.
2. Activer OAuth 2.0 avec Authorization Code + PKCE.
3. Ajouter le callback HTTPS de la Cloud Function X.
4. Copier le Client ID dans `social-config.js` et passer `configured: true`.
5. Définir côté Firebase `X_CLIENT_ID`, `X_REDIRECT_URI` et le secret `X_CLIENT_SECRET`.
6. Déployer les fonctions, puis utiliser **Connecter X**.

Le niveau de lecture et de publication dépend du niveau d’accès API X attribué à l’application.
