# Sigma V4.9.6 — LinkedIn Setup

1. Créer une application LinkedIn Developer.
2. Activer **Sign In with LinkedIn using OpenID Connect**.
3. Copier le Client ID dans `social-config.js` et passer `configured: true`.
4. Définir côté Firebase `LINKEDIN_CLIENT_ID`, `LINKEDIN_REDIRECT_URI` et le secret `LINKEDIN_CLIENT_SECRET`.
5. Déployer `functions/`, puis utiliser **Connecter LinkedIn**.

Le profil OpenID Connect est pris en charge. Les pages, publications et statistiques avancées restent soumises aux produits et autorisations LinkedIn accordés.
