# Sigma Life OS V4.9.3 — LinkedIn

Patch incrémental à appliquer après V4.9.2.

## Fonctionnel après configuration
- OAuth LinkedIn côté Cloud Functions ;
- import du profil via OpenID Connect ;
- stockage privé du jeton ;
- enregistrement dans le Social Engine ;
- diagnostic et déconnexion.

L'accès aux publications/pages/analytics dépend des produits LinkedIn et autorisations approuvés pour l'application.

## Secrets et paramètres Firebase
- secret : `LINKEDIN_CLIENT_SECRET`
- paramètres : `LINKEDIN_CLIENT_ID`, `LINKEDIN_REDIRECT_URI`

Ne jamais mettre le Client Secret dans GitHub.
