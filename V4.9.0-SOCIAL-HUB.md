# Sigma Life OS V4.9.0 — Social Hub Foundation

Cette version crée le socle commun des réseaux sociaux sans prétendre connecter des API qui ne sont pas encore configurées.

## Inclus
- Centre social unifié dans l'écran Social existant.
- Modèle local normalisé : SocialAccount, SocialPost, SocialMessage, SocialComment, SocialMetric, SocialNotification et ScheduledPost.
- Configuration centralisée dans `social-config.js`.
- Diagnostic sans boucle MutationObserver.
- Fichiers V4 (`v4-system.css` et `v4-system.js`) inclus pour éviter les erreurs 404.
- Cache-busting V4.9.0.

## État des plateformes
- YouTube : prêt à être relié au compte Google existant lors de V4.9.3.
- Meta / Instagram : V4.9.1, application Meta et backend OAuth nécessaires.
- LinkedIn : V4.9.2, application LinkedIn et backend OAuth nécessaires.
- TikTok et X : V4.9.3, accès développeur requis.

## Déploiement GitHub Pages
Remplacer le dépôt avec le contenu du dossier `sum-main`, en conservant vos vraies valeurs dans `firebase-config.js`, `google-cloud-config.js` et `platform-config.js`.

Après le déploiement : attendre GitHub Pages, puis faire Ctrl+Shift+R.
