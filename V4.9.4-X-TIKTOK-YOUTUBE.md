# Sigma Life OS V4.9.4 — X, TikTok et YouTube

Patch incrémental à appliquer après V4.9.3.

## YouTube
Utilise la connexion Google existante et injecte les abonnements/métriques dans le Social Engine.

## X
OAuth 2.0 Authorization Code avec PKCE côté Cloud Functions, profil et métriques publiques. Les niveaux d'accès X peuvent limiter les endpoints.

## TikTok
Login Kit côté Cloud Functions, profil et métriques de base. L'accès aux vidéos/publication dépend des produits et validations activés dans TikTok for Developers.

## Secrets et paramètres
- X : `X_CLIENT_SECRET`, `X_CLIENT_ID`, `X_REDIRECT_URI`
- TikTok : `TIKTOK_CLIENT_SECRET`, `TIKTOK_CLIENT_KEY`, `TIKTOK_REDIRECT_URI`

Ne jamais publier les secrets dans GitHub.
