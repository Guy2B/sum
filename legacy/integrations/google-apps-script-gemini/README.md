# Gemini proxy gratuit (Google Apps Script)

1. Créez un projet Apps Script.
2. Collez `Code.gs`.
3. Paramètres du projet → Propriétés du script → ajoutez `GEMINI_API_KEY`.
4. Déployer → Nouveau déploiement → Application Web.
5. Exécuter en tant que vous ; accès selon votre politique de bêta.
6. Copiez l'URL `/exec` dans `google-cloud-config.js` → `appsScriptAiProxyUrl`.

Ne placez jamais la clé Gemini dans GitHub Pages.
