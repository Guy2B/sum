# Déploiement V3.0

1. Remplacer tout le dépôt GitHub par le contenu de cette archive.
2. Vérifier `firebase-config.js`, `google-cloud-config.js` et `platform-config.js`.
3. Attendre GitHub Pages.
4. Ouvrir DevTools > Application > Service Workers, désinscrire l’ancien worker et effacer les données du site.
5. Recharger avec Ctrl+Shift+R.
6. Se connecter au compte Sigma, ouvrir Messagerie et cliquer Google.
